# [Problem](https://leetcode.com/problems/minimum-ascii-delete-sum-for-two-strings/)

#### DP
````
class Solution {
    public int minimumDeleteSum(String s1, String s2) {
        int m = s1.length();
        int n = s2.length();
        int[][]dp = new int[n+1][m+1];
        
        for(int i = 1; i <= m ; i++) dp[0][i] = dp[0][i-1] + (s1.charAt(i-1)-0);
        for(int i = 1; i <= n ; i++) dp[i][0] = dp[i-1][0] + (s2.charAt(i-1)-0);
        for(int i = 1; i <= n; i++){
            for(int j = 1; j <= m; j++){
                if(s1.charAt(j-1) == s2.charAt(i-1)){
                    dp[i][j] = dp[i-1][j-1];
                }else{
                    dp[i][j] = Math.min(dp[i][j-1]+(s1.charAt(j-1)-0), dp[i-1][j] + (s2.charAt(i-1)-0));
                }
            }
        }
        return dp[n][m];
    }
}
````
