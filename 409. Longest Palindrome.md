#[Problem](https://leetcode.com/problems/longest-palindrome/)

### ASCII
```
    public int longestPalindrome(String s) {
        int[] count = new int[128];
        int unique = 0;
        int result = 0;
        for(char d:s.toCharArray()){
            if(count[d] == 0){
                unique++;
                count[d]++;
            }else{
                result += 2;
                unique--;
                count[d]--;
            }
        }
        if(unique != 0) result++;
        return result;
    }
```
