### Math(Hash)
```
    public String convertToTitle(int n) {
        StringBuilder sb = new StringBuilder();
        while(n > 26){
            int m = n % 26;
            if(m == 0){
                sb.append('Z');
                n = n - 26;
            }else{
                sb.append((char)('A'+m-1));                
            }
            n = n / 26;
        }
        sb.append((char)('A'+n-1));
        return sb.reverse().toString();
    }
```
