# [Problem](https://leetcode.com/problems/find-pivot-index/)
### 
```
    public int pivotIndex(int[] nums) {
        int right = 0;
        for(int i = 1;i<nums.length;i++){
            right += nums[i];
        }
        int left = 0;
        for(int i = 0;i<nums.length;i++){
            if(right == left){
                return i;                
            }else if(i == nums.length-1){
                break;  
            } else{
                left += nums[i];
                right -= nums[i+1];
            }
        }
        return -1;
    }
```
