# [Problem](https://leetcode.com/problems/continuous-subarray-sum/)


#### BruteForce
````
class Solution {
    public boolean checkSubarraySum(int[] nums, int k) {
        for(int i = 0; i < nums.length -1 ; i++){
            int sum = nums[i];
            for(int j = i + 1; j < nums.length; j++){
                sum += nums[j];
                if((k == 0 && sum == 0) || (k != 0 && sum % k == 0)) return true;
            }
        }
        return false;
    }
}
````
