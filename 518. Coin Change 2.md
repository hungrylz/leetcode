# [Problem](https://leetcode.com/problems/coin-change-2/)


#### DP
````
class Solution {
    public int change(int amount, int[] coins) {
        int[] dp = new int[amount+1];
        dp[0] = 1;
        
        for(int d : coins){
            
            for(int i = 1; i <= amount; i++){
                if(i - d >= 0) dp[i] += dp[i-d];
            }
        }
        return dp[amount];
    }
}
````
