# [Problem](https://leetcode.com/problems/search-insert-position/)

#### Binary Search
````
/*

  1 3 5 6
   2     
    m  
    l 
  r
  
  
*/
````
````
    public int searchInsert(int[] nums, int target) {
        int l = 0;
        int r = nums.length-1;
        while(l<=r){
            int m = (l + r)/2;
            if(nums[m] == target) return m;
            if(nums[m] > target){
                r = m - 1;
            }else{
                l = m + 1;
            }
        }
        return r+1;
    }
````
